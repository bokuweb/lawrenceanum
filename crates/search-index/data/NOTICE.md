# jp-law-thesaurus.json について

法律ターム↔同義語の静的辞書。検索インデックス時に本文へ出現した term の同義語を
追記し、表現ゆれ（例: 「BIS規制」↔「バーゼル規制」）を吸収して recall を上げる。

- 出自: ellisii-toolkit `crates/jp-law-thesaurus`（同一作者）。

  本プロジェクトを商用提供する場合は作者として別途許諾/再ライセンスが必要
  （元データ出典は政府標準利用規約=CC BY 4.0相当の法令用語等を curate）。
- 更新時は ellisii-toolkit 側を正とし、本ファイルへコピーする。
